Here are some examples of diagrams build with DiagrammeR:

library(tidyverse)
library(DiagrammeR)
grViz("
digraph boxes_and_circles {
      graph [layout = circo,
             overlap = true]

      node [shape = circle,
            fixedsize = true,
            fontname = Helvetica,
            width = 1]
      Problem; Plan; Data; Analysis; Conclusion
      

      edge [color = grey]      
      Problem -> Plan
      Plan -> Data
      Data -> Analysis
      Analysis -> Conclusion
      Conclusion -> Problem
      }
      ")

This diagram is inspired by Figure 1 from the paper of Wild and Pfannkuch, 1999.
